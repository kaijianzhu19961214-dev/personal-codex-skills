# Awesome Human Distillation

Use this skill as a local index for human-distillation skill projects. It is a catalog, not a single executable workflow.

中文说明：这是 `awesome-human-distillation` 的个人仓库版入口，用来检索和比较“人物/经验/思维框架蒸馏成 skill”的项目。它本身不是某一个 persona skill，而是目录索引。

## Workflow

1. Read `references/catalog.md` for the Chinese catalog.
2. Read `references/catalog-en.md` only when English names/descriptions are useful.
3. Extract candidate repositories that match the user's interest.
4. Before installing any listed skill, use `skill-vetter` to inspect source, license, scripts, prompts, and Codex compatibility.
5. Treat persona/digital-twin skills as experimental unless the source and consent model are clear.

## Output

When recommending candidates, include:

- skill/project name
- repository URL
- why it is relevant
- likely use case
- risk notes, especially privacy, consent, copyright, licensing, or prompt quality

Do not install listed projects automatically unless the user explicitly asks.
